# AAA Registry

This repo hosts the public pack registry index and template registry.

## Purpose / Scope
Provide the single source of truth for AAA registry indexes and template catalogs.

## Ownership / CODEOWNERS
Ownership is defined in CODEOWNERS at the repo root.

## Versioning / Release
Registry updates are versioned through Git history and published via releases.

## How to Consume / Use
Consumers should read `registry_index.json` and `template_registry.json` for current assets.

## Contribution / Promotion Rules
Registry entries must be reviewed and validated before publication.

- `registry_index.json` is the source of truth.
- `template_registry.json` indexes reusable templates.
- Packs are published as GitHub release artifacts.
